# RWSCargo: дорожная карта

## Статус на 19.06.2026

Проект уже не на нулевом этапе. Есть рабочий Astro-сайт, SEO-база, Docker-сборка и первый вариант иммерсивного hero.

## Этап 1. Контекст и направление

Статус: выполнено.

Сделано:

- сохранены референсы в `docs/references/`;
- зафиксирован B2B-фокус: крупный бизнес, коммерческие грузы, доставка "в белую";
- выбран стек Node.js + Astro + TypeScript;
- заложены Docker, PostgreSQL и Redis как optional infra;
- создана проектная документация.

## Этап 2. Техническая основа

Статус: выполнено.

Сделано:

- инициализирован Astro-проект;
- подключен TypeScript check;
- настроен `scripts/run-astro.mjs`;
- добавлены Dockerfile и `docker-compose.yml`;
- сайт собирается через `npm run build`;
